Quiz Game !

  • • Tekan tombol Start Quiz untuk memulai.
  • • Pertanyaan pertama ditampilkan, timer mulai dari 15 detik.
  • • Tekan Submit atau tombol Enter untuk mengirim jawaban.
  • • Jawaban tidak dapat diubah setelah dikirim atau waktu habis.
  • • Setiap pertanyaan memiliki waktu 15 detik.
  • • Jika waktu habis, pertanyaan berikutnya muncul setelah 3 detik.
  • • Tekan tombol Next atau tombol 'ArrowRight' untuk melanjutkan ke pertanyaan berikutnya.
  • • Jika semua pertanyaan dijawab, skor akhir ditampilkan dan kuis berakhir.
